## Queue Rules

- Do not publish more than one page per day.
- Upgrade before creating new URLs.
- A page can move to `qa` only when the proof variable is written into the content.
- A page can move to `scheduled` only after build and render checks.
- A page can move to `gsc_checked` only after live URL is inspected or logged for Search Console follow-up.
